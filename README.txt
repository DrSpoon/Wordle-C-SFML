Instructions:
1. You have to guess the Wordle in six goes or less.
2. A correct letter turns green.
3. A correct letter in the wrong place turns yellow.
4. An incorrect letter turns gray.

How do I play?

Press on the box and write a character.

Be careful you can not change it afterwards!







The game is written in C++ with the SFML library if you encounter problem that is because 
I am working in Code::Blocks IDE and a lot of things don't work.
